ClipStackClient プライバシーポリシー

ClipStackClient は、macOS 版 ClipStack の連携アプリとして設計されています。
このページでは、現在のリリースにおける iPhone/iPad 上でのデータの扱いを説明します。

現行ポリシー

概要

ClipStackClient はユーザーのプライバシーを尊重します。
アプリは、macOS 版 ClipStack が生成した暗号化済みの同期データを読み込み、 iPhone/iPad 上で同期テキスト履歴を閲覧・コピーできるようにします。

アプリが扱う情報

  • ClipStack から同期されたテキスト履歴
  • 同期データに含まれる各種メタデータ
  • iCloud Keychain 経由で利用可能になる同期用暗号鍵
  • 同期失敗時の fallback 用として保持される暗号化キャッシュ

現在の同期方式

現在のリリースでは、CloudKit private database 上の同期内容を読み込みます。
ClipStackClient は、開発者が運用するサーバーへ クリップボード履歴をアップロードしません。

ローカル保存

最新の同期内容を再表示できるように、 アプリは暗号化済み同期データのローカルキャッシュを保持する場合があります。

共有された暗号鍵は、端末上での復号のために利用されます。

クリップボードへの書き戻し

一覧項目をタップすると、選択したテキストが iPhone/iPad のシステムクリップボードへコピーされます。

データ収集

ClipStackClient はユーザーアカウントを必要としません。
現在のリリースには、解析 SDK、広告 SDK、 開発者運用のバックエンドサービスは含まれていません。

ユーザーの選択

  • Mac 側の ClipStack で ClipStackClient 同期をオフにする
  • iCloud Keychain を無効にする、または同期運用をやめる
  • Mac 側 ClipStack から同期内容を削除・変更する
