CopyPouch Pocket サポート

CopyPouch Pocket のローカル保存、初期設定、同期設定、 暗号鍵に関する補足情報です。

よくある質問

主なサポート項目

インストール直後に履歴が表示されない

Mac からの同期履歴は、CopyPouch 側で同期を有効にするまで表示されません。 ただし、CopyPouch Pocket 単体でもローカルクリップを保存して使えます。

同期履歴を利用する場合は、初回セットアップに従って、 Mac 側 CopyPouch との連携を行ってください。

iPhone / iPad 側だけでテキストを保存したい

メイン画面左上の + から New Clip を開くと、 現在のクリップボードや任意入力したテキストをこの端末に保存できます。

保存したローカルクリップは Source の Local で絞り込めます。 不要になった場合は、設定画面からローカルクリップを全削除できます。

iCloud が利用できないと表示される

Mac からの同期履歴を使う場合は、iCloud と iCloud Keychain を利用します。
端末で iCloud と iCloud Keychain が有効か、同期に使っている環境で同じ Apple ID が 使われているかを確認してください。

iPhone 側だけで自動的にクリップボード履歴を集めたい

現在のリリースでは、バックグラウンドでクリップボード履歴を自動収集する機能には対応していません。
残したいテキストは、New Clip 画面から手動でローカル保存してください。
