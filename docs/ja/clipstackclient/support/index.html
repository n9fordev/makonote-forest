ClipStackClient サポート

ClipStackClient の利用方法、初期設定、同期設定、 暗号鍵に関する補足情報です。

よくある質問

主なサポート項目

履歴が見れない・表示されない

ClipStackClient は、Mac で動いている ClipStack と連携して利用します。

初回セットアップに従って、初回起動時はClipStackとの連携(暗号鍵の登録)を行ってください。

iCloud が利用できないと表示される

現在のリリースでは、CloudKit 上の同期内容と iCloud Keychain の同期鍵を使います。
端末で iCloud と iCloud Keychain が有効か、同期に使っている環境で同じ Apple ID が 使われているかを確認してください。

iPhone 側だけでクリップボード履歴を集めたい

対応していません。ClipStackClientは Mac 側 ClipStack の履歴を閲覧・コピーする連携アプリです。
単体の iPhone クリップボード履歴アプリとしては動作しません。
