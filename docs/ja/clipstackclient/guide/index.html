ClipStackClient ガイド

このページでは、ClipStackClient を ClipStack と連携させる手順と、 iPhone / iPad 上での基本的な使い方をまとめています。

ClipStackClientは、MacとiOS間のコピーを快適に行うアプリです。

履歴の本体は Mac 側の ClipStack に置き、ClipStackClient は 必要なときにだけ開いて、同期済みクリップを探してコピーする モバイル向けの連携アプリとして使います。

⚠ 注意: ClipStackClient を使うには、macOS 版 ClipStack が必要です。
このアプリ単体では履歴を収集せず、Mac 側で作成・更新された内容を iPhone/iPad で利用するためのアプリとして動作します。

初回セットアップ

  1. Mac の ClipStack の設定画面で Sync with ClipStackClient をオンにする
  2. Mac と iPhone / iPad の両方で iCloud Keychain を有効にしておく
  3. メイン画面へ戻り、更新して最新の同期内容を読み込む

このスイッチをオフにすると、ClipStack は ClipStackClient との CloudKit 同期を停止します。

メイン画面の基本操作

  • 一覧行をタップするとそのままコピーできる
  • 検索バーで表示中の一覧を絞り込める
  • All / Pinned / URLs を切り替えられる
  • 画面を引き下げる操作と右上ボタンで更新できる

Pinned タブ

Pinned タブは、Mac 側でピン留めした項目だけを表示します。

URLs タブ

URLs タブは、URLと判定された項目だけを表示します。

同期ステータスの見方

  • Up to date: 最新の同期が成功している状態
  • Syncing: CloudKit 上の最新内容を確認している状態
  • Sync failed: 最新の同期内容を読めなかった状態
  • Sync key unavailable: 共有された同期鍵がまだ届いていない状態
現在のリリースでは、iPhone 側から履歴を編集しません。
ピン留め変更、削除、同期ファイルの更新は Mac 側 ClipStack で行ってください。

設定項目

  • 共有された同期鍵が利用可能か確認する
  • 現在の同期元と最終同期時刻を確認する
  • サポート情報とプライバシーポリシーを開く
