ClipStackClient

ClipStackClient は、iPhone / iPad 上でテキストをローカル保存し、すばやく再利用できるアプリです。
Mac 側の ClipStack と連携すると、同期されたテキスト履歴も同じ画面から確認できます。

使い方 サポート プライバシーポリシー

概要

ClipStack の流れをモバイルへ広げる、軽量な連携アプリです。

iPhone / iPad 上で残しておきたいテキストを保存しながら、
Mac 側でコピーしたアイテムもシームレスに再利用できます。

  • 1 タップですぐコピー
  • macOS 版 ClipStacks からの自動同期
  • All / Pinned / URLs 機能
  • 履歴の検索
⚠ 注意: macOS 版 ClipStacks との同期機能を利用するためには
iPhone / iPad 上で Mac側と同じ iCloud アカウントでログインする必要があります。
