ClipStackClient

ClipStackClient は、macOS 版 ClipStack のための無料連携アプリです。
同期されたテキスト履歴を iPhone / iPad で確認し、そのまま 1 タップでコピーできます。

使い方 サポート プライバシーポリシー

概要

ClipStack の流れをモバイルへ広げる、軽量な連携アプリです。

Mac 側でコピーしたアイテムを、必要なときだけ iPhone/iPad で開き、スムーズにコピーすることができます。

  • 1 タップですぐコピー
  • macOS 版 ClipStack からの自動同期
  • テキスト内容を暗号化して同期
  • All / Pinned / URLs 機能
  • 履歴の検索
⚠ 注意: ClipStackClient を使うには、macOS 版 ClipStack が必要です。
このアプリ単体では履歴を収集せず、Mac 側で作成・更新された内容を iPhone/iPad で利用するためのアプリとして動作します。
