ClipStack プライバシーポリシー

ClipStack はローカル完結を前提とした クリップボードユーティリティです。
このページでは、現在のリリースにおける データの扱いを説明します。

現行ポリシー

概要

ClipStack はユーザーのプライバシーを尊重します。
クリップボード履歴は、アプリの基本機能を提供するために デバイス上へローカル保存されます。

取り扱う情報

  • テキスト
  • リンク
  • 画像

これらの情報は、アプリ内で クリップボード履歴機能を提供する目的に限って扱われます。

ローカル保存

クリップボード履歴はユーザーのデバイス上に保存されます。
履歴はいつでも削除でき、 設定から永続保存を無効にすることもできます。

データ収集

ClipStack はユーザーアカウントを必要としません。
現在のリリースでは、 クリップボード履歴をリモートサーバーへ送信したり、 デバイス間で同期したりしません。

権限

ClipStack は、選択した履歴項目を 利用中のアプリへ直接貼り付けるために、 macOS のアクセシビリティ権限を要求する場合があります。

この権限を付与しない場合でも、 履歴の閲覧、検索、 クリップボードへの再コピーは行えます。

第三者提供

ClipStack は、ローカル完結の基本動作として、
クリップボード履歴を第三者と共有しません。

ユーザーの選択

  • 履歴の永続保存を無効にする
  • 個別の履歴項目を削除する
  • すべての履歴を削除する
  • システム設定からアクセシビリティ権限を取り消す
