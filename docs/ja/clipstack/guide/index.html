ClipStack 使い方ガイド

ClipStack を初めて使う方向けに、起動方法、権限設定、基本操作、設定項目をまとめています。

ClipStack は、必要なときだけ開いて使うクリップボード履歴ツールです。

普段はメニューバーに常駐し、必要なタイミングでパネルを開いて、 コピー済みのテキスト、リンク、画像を再利用する使い方を想定しています。

起動方法

  • メニューバーのアイコンをクリックして開く
  • グローバルホットキーで開く

初期設定のホットキーは Control + Option + Command + V です。 ホットキーは設定画面から変更できます。

アクセシビリティの設定

他アプリへ直接貼り付けるには、macOS のアクセシビリティ権限が必要です。 初回利用時や未許可の状態では、ClipStack に案内が表示されます。

設定画面が開いたら、プライバシーとセキュリティ の アクセシビリティ で ClipStack を許可してください。

基本操作

  • 項目をクリックすると、その内容を再利用できます
  • テキストやリンク・画像は、利用中のアプリへ直接貼り付けます
  • 直接貼り付けできない場合でも、選択した内容はクリップボードへ戻されます

キーボード操作

Control + Option + Command + V ClipStack のパネルを開きます
Esc パネルを閉じます
Enter 選択中の項目を再利用します
↑ / ↓ 履歴項目を移動します
← / → タブを切り替えます
Command + F 検索欄へ移動します

各タブの見方

  • Recent: 直近の履歴を表示します
  • Pinned: ピン留めした項目だけを表示します
  • Links: URL として判定された項目だけを表示します
  • Images: 画像として取り込まれた項目だけを表示します

検索

上部の検索欄では、表示中のタブに対して絞り込みが行われます。 たとえば Links タブで検索すれば、リンクだけを対象に絞り込めます。

ピン留めと削除

よく使う項目はピン留めして残しておけます。不要な項目は、 スワイプ操作でその場で削除できます。

右にスワイプすると Pin / Unpin、左にスワイプすると Delete を実行できます。

スワイプしにくい場合は、右クリックメニューから操作することもできます。

画像のプレビュー

画像項目のプレビューアイコンにマウスを乗せると、 簡易プレビューが表示されます。貼り付けや再利用の前に、 内容を確認したいときに便利です。

リンクを開く

URL として判定された項目は、右側のアイコンや右クリックメニューから そのままブラウザで開けます。貼り付けるだけでなく、 すぐ確認したいリンクを開く用途にも使えます。

権限がない場合でも、ClipStack は内容をクリップボードへ戻し、 手動貼り付けできる状態までは維持します。

設定で変更できること

  • 履歴を再起動後も保存するかどうか
  • 保存件数の上限（20 / 50 / 100 / 200）
  • グローバルホットキーのプリセット
  • 履歴の全削除
