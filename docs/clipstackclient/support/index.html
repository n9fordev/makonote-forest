Frequently asked questions

Common support topics

No clips appear after installing the app

ClipStackClient depends on a Mac already running ClipStack.

First, make sure ClipStack on macOS is set to sync with ClipStackClient through CloudKit. Then update the main screen in ClipStackClient after iCloud Keychain has shared the sync key.

The app says the encryption key is missing

ClipStackClient needs the shared sync key from iCloud Keychain.

Make sure both devices use the same Apple Account and have iCloud Keychain turned on. Then open ClipStackClient again and refresh.

The app says iCloud is unavailable

The current release reads synced history from CloudKit and waits for the sync key through iCloud Keychain.
Make sure iCloud and iCloud Keychain are enabled on the device and that the same Apple Account is being used for the synced environment.

Can ClipStackClient collect iPhone clipboard history by itself?

No. The current release is a viewer and copy app for the Mac app only.
It does not run as a standalone iPhone clipboard recorder.
