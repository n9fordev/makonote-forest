Frequently asked questions

Common support topics

No synced clips appear after installing the app

Synced Mac history appears after CopyPouch on your Mac has sync turned on. You can still save local clips in CopyPouch Pocket without setting up Mac sync.

First, make sure CopyPouch on macOS is set to sync with CopyPouch Pocket. Then update the main screen in CopyPouch Pocket after iCloud Keychain has shared the sync key.

How do I save text only on my iPhone or iPad?

Tap + on the main screen to open New Clip. You can save the current clipboard text or type custom text.

Local clips can be filtered with Source > Local. If needed, you can delete all local clips from Settings.

The app says the encryption key is missing

CopyPouch Pocket needs the shared sync key from iCloud Keychain.

Make sure both devices use the same Apple Account and have iCloud Keychain turned on. Then open CopyPouch Pocket again and refresh.

The app says iCloud is unavailable

Synced Mac history uses iCloud and iCloud Keychain.
Make sure iCloud and iCloud Keychain are enabled on the device and that the same Apple Account is being used for the synced environment.

Can CopyPouch Pocket collect iPhone clipboard history automatically?

Not in the current release. CopyPouch Pocket does not collect clipboard history in the background.
Save text manually from the New Clip screen when you want to keep it.
