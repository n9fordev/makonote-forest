ClipStack Privacy Policy

ClipStack is designed as a local-first clipboard utility.
This page describes how the current release handles data on macOS.

Current policy

Overview

ClipStack respects your privacy.
Clipboard history is stored locally on your device to provide the app’s core functionality.

Information we handle

  • Text
  • Links
  • Images

This content is used only to provide clipboard history features inside the app.

Local storage

Clipboard history is stored locally on your device.
You can clear history at any time, and you can disable persistent history in Settings.

Data collection

ClipStack does not require a user account.
The current release does not upload clipboard history to a remote server and does not sync clipboard history across devices.

Permissions

ClipStack may request Accessibility access on macOS so it can
paste a selected history item directly into the app you were using.

If you do not grant this permission, you can still browse history, search items, and copy them back to the clipboard.

Sharing

ClipStack does not share clipboard history with third parties
as part of its core local-only behavior.

Your choices

  • Disable persistent history
  • Delete individual history items
  • Clear all history
  • Revoke Accessibility access in System Settings
