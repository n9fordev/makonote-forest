ClipStacks

ClipStacks is a clipboard history utility for macOS.
It keeps recent text, links, and images available from a single panel.

How to use Support Privacy Policy

Overview

A practical clipboard history tool for everyday use.

ClipStacks is designed around a simple workflow:
open the panel, identify the item you want, and return to the current task without unnecessary steps.

  • Search recent clipboard items from a keyboard-first panel
  • Reuse copied text, links, and images
  • Pin important snippets and keep links and images easy to find
  • Open the panel from the menu bar or a global shortcut
  • Preview images before reusing them
  • Store history locally on your Mac
Selecting an item copies it back to the clipboard so you can paste it into the app you are already using.
