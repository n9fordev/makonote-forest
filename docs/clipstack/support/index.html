Frequently asked questions

Common support topics

Direct paste does not work

ClipStack needs Accessibility access to paste directly into another app.

Open System Settings > Privacy & Security > Accessibility, then allow ClipStack.

Screenshots do not appear in history

ClipStack can import screenshots only when they are copied to the clipboard as images.
Screenshots saved only as files are not imported automatically.

Where is clipboard history stored?

Clipboard history is stored locally on your Mac.
The current release does not require an account and does not sync clipboard history to a server.
