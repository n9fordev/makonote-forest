Frequently asked questions

Common support topics

How do I reuse an item?

Selecting an item copies it back to the clipboard. Paste it into the app you are using with Command-V.

Screenshots do not appear in history

CopyPouch can import screenshots only when they are copied to the clipboard as images.
Screenshots saved only as files are not imported automatically.

Where is clipboard history stored?

Clipboard history is stored locally on your Mac.
The current release does not require an account and does not sync clipboard history to a developer-operated server.

How does CopyPouch Pocket sync work?

CopyPouch can upload encrypted text history to CloudKit for CopyPouch Pocket.

Turn on Sync with CopyPouch Pocket in Settings, and keep iCloud Keychain enabled so CopyPouch Pocket can read the synced contents.
